PROGRAMMING LEARNING WEBSITE
=============================

This is a fully responsive educational website for learning programming languages, built with vanilla HTML, CSS, and JavaScript.

FEATURES
--------
1. Modern, clean UI with soft, pleasing colors suitable for learning
2. Light/dark mode toggle with localStorage persistence
3. Fully responsive design for desktop, tablet, and mobile
4. Search functionality for programming languages
5. Language cards with details and links to learning pages
6. Educational comparison guides for choosing languages
7. Semantic HTML and accessible design

FILE STRUCTURE
--------------
- index.html: Main homepage
- style.css: All styles for the website
- script.js: All JavaScript functionality
- language-pages/: Directory containing individual language pages
  - html.html: HTML learning page
  - css.html: CSS learning page
  - javascript.html: JavaScript learning page
  - python.html: Python learning page

HOW TO USE
----------
1. Open index.html in any modern web browser
2. Use the search bar to find specific languages
3. Click on language cards to navigate to learning pages
4. Toggle between light and dark mode using the moon/sun icon
5. On mobile, use the hamburger menu for navigation

EXPANDING THE WEBSITE
---------------------
To add more language pages:
1. Create a new HTML file in the language-pages directory
2. Follow the structure of existing language pages
3. Update navigation links as needed
4. Add the new language card to index.html

To add more content:
1. The CSS is organized with CSS custom properties for easy theming
2. The JavaScript is modular and well-commented
3. All pages follow semantic HTML structure

BROWSER COMPATIBILITY
---------------------
The website is compatible with:
- Chrome 60+
- Firefox 55+
- Safari 10+
- Edge 79+

All code is written with modern web standards and includes appropriate fallbacks.
