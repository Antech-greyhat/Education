About

AntechLearn is a full‑stack educational platform. This backend API provides secure authentication, user management, course handling, and administrative controls.

Endpoints

  • /register – User registration
  • /login – Authentication
  • /forgot-password – Reset workflow
  • /admin – Admin dashboard
  • /newsletter – Subscription management
  • /contact – Contact form handling

Explore full documentation at /docs.
