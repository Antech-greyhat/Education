Antech API

Powerful, secure, and scalable backend for modern educational platforms. Built with Flask-RESTX and designed for performance.

API Docs GitHub

About AntechLearn

AntechLearn is a full-stack educational platform. This backend API provides secure authentication, user management, course handling, and administrative controls — ready to power your learning app.

Key Endpoints

/register

Create a new user account. Returns JWT token on success.

/login

Authenticate existing user. Returns access token.

/forgot-password

Request a password reset email. Secure token flow.

/admin

Admin dashboard endpoints (protected, role-based).

/newsletter

Subscribe/unsubscribe from newsletters.

/contact

Submit contact form messages.

Explore full documentation at /docs

Tech Stack

Flask Flask-RESTX SQLAlchemy Alembic JWT Flask-Mail PostgreSQL Redis Gunicorn

Quick example (login) 


          
curl -X 'POST' \
  'https://antechapi.vercel.app/auth/login' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "email": "user@gmail.com",
  "password": "********"
}'

Returns a JWT token for subsequent requests.
